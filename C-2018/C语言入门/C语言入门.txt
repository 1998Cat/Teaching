1、程序设计
    由计算机开发人员按照做事情的步骤与方法编写计算机能够识别的一系列有序指令的集合

    程序 = 算法 + 数据结构

2、算法
    完成事情的方法（操作）与步骤

    基本特征：
        1）有穷性（一定会结束）
        2）确定性
        3）有0个或多个输入
        4）有1个或多个输出
        5）有效性

3、流程
    取钱示例：
        1）插入银行卡
        2）输入密码，如果密码正确，进入到第3步，如果密码错误，进入到第4步
        3）点击取款操作，进入到第5步
        4）提示密码错误，回到第2步
        5）输入取款金额
        6）判断取款金额是否小于存款金额，如果小于存款金额，进入到第7步，否则进入到第8步
        7）取走现金，进入到第9步
        8）提示存款额不足，回到第5步
        9）点击退卡
        10）取走银行卡

6、C语言程序设计
    程序设计：用编程语言来制定一系列有序指令的集合
    编程语言：
        1）低级语言
            最低级的编程语言：只包含0，1
            汇编语言：mov, alloc英文单词，直接操作内存空间（内存地址）
        2）高级语言
            利用人类自然语言来编写计算机程序（英文），利于人类阅读、理解、编写
            C语言，Java, C++, Basic, VB, C#(C Sharp), PHP, Python, JavaScript, Swfit, Objective-C

    基本介绍：
        1）1972年创办的
        2）主要用于操作系统，解释器，编译器，汇编器等系统级软件的开发（Unix, Linux, Windowx）
        3）1990,ISO国际化标准认证

    基本特征：
        简洁
        功能强大
        性能好
        移植性好
        在高级语言当中属于比较低级的语言
        是一门结构化编程语言

9、C语言程序的执行过程
    1）编写包含有C程序的源代码
    2）把源代码文件通过C语言的编译器转成低级语言，生成.obj(.o)文件
        编译器：把高级语言转化成低级语言（汇编语言）
        解释器：从上往下一行一行代码进行解释执行
    3）把转化成的汇编语言与系统或语言提供的其他的库进行链接操作，链接完后生成一个.exe的可执行文件

    .c  ----->   .obj     ----->    .exe

    常见的C/C++编译器：
        turbo c
        vc++(Microsoft Visual C++ 6.0)
        GCC(开源的编译器，Linux操作系统中经常使用此编译器)

10、C语言程序的结构
    1）预处理指令
        以#号开头的代码
            a、#include 包含头文件（引用一个已经写好的C程序文件），.h
            任何一个C语言程序都必须包含stdio.h这个文件

            #include "stdio.h"  ""可以引用编译器include目录中的.h文件，还可以引用开发人员自己定义的.h文件
            #include <stdio.h>  <>一般只能引用编译器include目录中的.h文件

            b、#define

    2）函数（处理事情的步骤集合）
        void main() {}

        main函数，名字是固定的，并且是整个C程序的入口函数，一个正常跑起来的C程序有且只有一个main函数，如果没有main函数，则程序无法运行

        void: 返回值，空
        main：函数名
        ()：定义形式参数
        {}： 函数体，函数里面所有的执行步骤

    3）函数体
        函数里面所有的执行步骤
        每一个执行步骤用英文分号结束，一个步骤最好用一行

11、C程序的编码规范
    1）严格区分大小写
    2）严格的缩进，缩进4个空格
    3）函数体的{}的摆放：第一种，直接在函数的()后面开始；第二各，换一行开始（推荐使用）
    4）见名知义
    5）关键字：C语言里保留的常用词，void, main, include……
    6）多余的行，空格都不需要，所有代码中能够影响程序运行的符号都用英文符号
    7）适当的注释：
        a、单行注释 // 注释内容
        b、多行注释
            /*
                注释内容，可以多行，没有必要去嵌套其他注释
            */
