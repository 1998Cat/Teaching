1、C语言的入口函数
    任何一个C语言程序中，都必须有main函数，此函数为整个程序的入口

    #include <stdio.h>

    void main()
    {

    }

    #include <stdio.h>

    int main()
    {
        return 0;
    }

2、函数的概念
    函数即是一个包含有具体算法实现的一个代码块
    函数的组成部分：
    
    返回值数据类型 函数名()
    {
        // 函数体
    }

    void main()
    {

    }

    返回值：void 表示空，返回空值
    函数名：main
    函数体：在{}的内部的代码都是函数体


    int main()
    {
        return 0;
    }

    返回值：int, 整体函数执行完毕后，需要返回一个整型数据
    函数名：main
    函数体：在{}的内部的代码都是函数体，如果有返回值，则在整个函数体适当的位置需要return语句

3、函数的调用
    在一个函数中去使用另外一个函数

    通过函数名来引用另外一个函数

    函数的返回值：被调用者执行完毕后，需要把一个值返回给调用者

4、函数定义与调用注意事项
    1）函数是否需要有返回值，如果需要返回值，则在函数名的前面加上返回值的数据类型，如果说不需要返回值，则函数名前面添加void
    2）函数名的命名规则：
        a、不能使用关键字
        b、只能是英文字母，下划线，数字，且只能是下划线和字母开头
        c、如果出现多个单词，用下划线分割多个单词
        d、见名知义
    3）如果返回值不为void，则函数体内部必须得有return语句，如果返回值为void，则不需要return语句，但是默认情况下是return;
    4）一个函数碰到return语句，则整个函数结束
    5）return返回的数据必须与函数名前返回值的类型一致
    6）一个程序中可以有多个自定义函数，但是不能重名

    函数的定义与函数的声明：
    1）函数的定义：包含了函数体
        int sum()
        {
            // 代码
            return 0;
        }
    2）函数的声明：不包含函数体
        int sum();

    如果函数的定义写在main函数之前，则可以直接调用

    如果函数的定义写在main函数后面，则需要在main函数之前声明自己的函数（方便开发者知道整个程序包含有哪些自定义函数）

5、函数的参数
    int getMoney(int money)
    {
        printf("%d", money);
    }

    getMoney(200); // int money = 200;
    getMoney(500); // int money = 500;
    getMoney(1000); // int money = 1000;

    double pow(int a, int b)
    {
        double x = 1.0;
        for (int i = 1; i <= b; i++)
        {
            x *= a;
        }
        return x;
    }

    pow(2, 2); // int a = 2, int b = 2
    pow(3, 5); // a = 3, b = 5

    返回值类型 函数名(形参列表)
    {
        // 函数体
    }

    形参列表：
        参数数据类型 参数名, 参数数据类型 参数名, ......

        参数的数据类型是必须的
        参数名由开发者自定义，要有意义
            1）在函数定义中，参数名是必须的
            2）在函数声明中，参数名可以省略，也可以保留，并且名称可以与定义中的名称不一样
        
        在函数定义中，参数列表称为形式参数列表（形参列表），仅仅做声明，并且在函数体中可以当作变量来使用
            形参列表在函数体内部的使用方法跟变量的使用方法完全一致
        在函数调用中，参数列表称为实际参数列表（实参列表），给形参传递真实的值，在函数体中使用的时候，参数对应的变量就有具体的值

        实参顺序与形参顺序保持一致，数据类型也要保持一致

6、为什么使用函数？
    1）简化代码的编写
    2）增加代码的复用性
    3）增加代码的可读性
    4）自顶向下，逐步求精思想的使用
